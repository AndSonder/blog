# PFCC blog

PFCC 社区博客

## 本地开发

请先确保本地安装好 [Node.js](https://nodejs.org/) 18+，并安装 [pnpm](https://pnpm.io/) 作为包管理器。

之后运行如下命令安装所有依赖项

```bash
pnpm install
```

安装完成后，运行如下命令启动本地开发服务器

```bash
pnpm dev
```

此时便可以按照提示打开相应的网址进行本地开发啦～

更多开发指南见 [CONTRIBUTING.md](./CONTRIBUTING.md)
